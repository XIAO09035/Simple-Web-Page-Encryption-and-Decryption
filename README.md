# Simple-Web-Page-Encryption-and-Decryption
# 🔐 密文网页解码工具

一个纯前端工具，用于解密经过字节取反的加密 ZIP 包，并在浏览器中通过虚拟服务器完整预览内部网页。

## ✨ 功能特点

- **字节取反解密**  
  对文件前 `n` 字节执行按位取反（`~byte & 0xFF`），还原原始 ZIP 数据。密码 `n` 可自定义（默认 `4096`）。

- **虚拟站点托管**  
  利用 Service Worker 将解密后的 ZIP 内容挂载到 `/decrypted/` 路径下，使内部网页的相对链接（CSS、JS、子页面）均可正常访问。

- **文件列表预览**  
  解密后列出压缩包内所有文件，每个文件支持：
  - **打开**：在新标签页通过虚拟路径预览
  - **下载**：直接保存单个文件到本地

- **完整包下载**  
  一键导出解密后的 ZIP 压缩包，方便离线保存。

- **纯本地运行**  
  所有解密与解压操作均在浏览器内完成，文件不会上传至任何服务器。

## 📦 使用方法

1. 将本 HTML 文件放置于 Web 服务器目录（如 IIS、Nginx）。
2. 将加密文件命名为 `encrypted.bin`（或手动选择），与 HTML 同目录。
3. **必须通过 HTTPS 或 `localhost` 访问页面**（Service Worker 要求安全上下文）。
4. 输入密码 `n`（默认 `4096`），点击「解密并挂载虚拟站点」。
5. 解密成功后，从文件列表中选择文件打开或下载。

## ⚙️ 加密方式说明

原始文件夹（例如包含 `index.html` 及资源）应打包为 **ZIP 格式**，然后对 ZIP 文件的前 `n` 个字节进行**按位取反**操作。

示例伪代码：
```javascript
for (let i = 0; i < n; i++) {
    data[i] = ~data[i] & 0xFF;
}
